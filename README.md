# Student MIS Portal

## 📌 Project Description
Student MIS (Management Information System) is a web-based application developed using HTML, CSS, JavaScript, bootstrap, PHP, and MySQL.  
The system helps students manage their profiles, login ,logout, view attendance ,mark attendance with there location, and access important information online.

## 🚀 Features
- Admin Login System
- Student Login System
- Admin Dashboard
- Add New Students
- Mark Student Attendance
- View Attendance Records
- Student Dashboard to Check Attendance
- Secure Password Change Option


## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Bootstrap
